# Audit ReportsThis directory contains AI Guard audit reports for test files.## Files- `audit_report_YYYYMMDD_HHMMSS.md` - Timestamped Markdown reports- `audit_report_YYYYMMDD_HHMMSS.json` - Timestamped JSON reports- `latest.md` - Latest audit report in Markdown format- `latest.json` - Latest audit report in JSON format## Report ContentsEach audit report includes:- **Summary Statistics**: Total files scanned, issues found by category- **Critical Secrets**: Hardcoded passwords, API keys, tokens- **Playwright Antipatterns**: Flaky test patterns (waitForTimeout, etc.)- **Bad Practices**: TODOs, console.logs, skipped tests## Viewing Reports**Markdown (human-readable):**```bashcat audit/latest.md```**JSON (machine-readable):**```bashcat audit/latest.json```## Security Notice⚠️ **Do not commit these files to version control!**Audit reports may contain sensitive information like:- Hardcoded credentials- API endpoints- Internal system detailsThis directory is added to `.gitignore` by default.## Report RetentionConsider implementing a cleanup policy:```bash# Keep only last 10 reportscd auditls -t audit_report_*.md | tail -n +11 | xargs rm -fls -t audit_report_*.json | tail -n +11 | xargs rm -f```
